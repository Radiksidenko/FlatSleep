# FlatSleep

FlatSleep is a native SwiftUI sleep tracking and analytics application integrated with Apple HealthKit. It transforms raw sleep intervals into modern, continuous timelines, concentric activity rings, and an interactive rolling calendar designed following Apple's premium design principles.

## How it works

- **HealthKit Integration**: Automatically fetches sleep stages (Awake, Core, Deep, REM) from Apple Health / Apple Watch via background queries.

- **Continuous Timeline & Hypnogram**: Displays sleep data through custom-built graphs with smooth, continuous curves or flat

- **Tactile Scrubbing**: Users can touch and drag a finger across the graphs to activate a precise floating marker, showing exact stage details and timestamps in real-time.

- **Activity Rings Grid**: Each calendar day features a custom triple-nested `SleepRingView` visualizing individual goals achieved for Deep, Core, and REM sleep phases simultaneously.
